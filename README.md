# 🚀 InvestQuest: The Anti-Gravity Financial Navigator

InvestQuest is a gamified, AI-powered investment platform designed to turn the "scary" world of finance into a cosmic mission for students. Built for the **Karke Dikha Hackathon**, it combines real-time strategic coaching with a premium, space-age UI.

## 🌌 Project Overview
InvestQuest teaches financial literacy through **gamified micro-investing**. Students earn XP, build streaks, and unlock "Stellar Assets" while being coached by the **Astro-AI**—a real-time defensive engine that protects them from high-risk anomalies and behavioral pitfalls.

## ✨ Core Features
- **SYBIL AI Hub**: A dual-intelligence engine supporting Local LLMs (Ollama) and high-fidelity generative fallback for personalized financial strategy.
- **Cognitive Portfolio Clustering**: The engine segments users into *Beginner*, *Consistent*, and *Advanced* tiers, dynamically adapting the UI and investment recommendations.
- **Mission Orbits**: A high-impact visual interface for deploying capital into diversified risk-weighted sectors.
- **Wealth Strategy Board**: A grid-based matching game that simulates asset allocation (Stocks, BTC, Gold, Bonds).
- **Behavioral Anomaly Detection**: Subsecond detection of unusual spending spikes or harmful inactivity.
- **Goal Optimization AI**: Predictive modeling that calculates your "time-to-goal" based on current momentum and win rates.

## 🛠️ Technology Stack
- **Frontend**: React (Vite), Vanilla CSS (Glassmorphism & Particle FX), Framer-style animations.
- **Backend**: Node.js & Express.
- **AI Intelligence**: Ollama (Local Llama3 Support) + Custom Neural Logic.
- **Persistence**: Modular JSON-based state sync for hackathon speed and isolation.

## 🚀 Getting Started

### 1. Clone & Setup
```bash
git clone https://github.com/poojitha813/the_logits.git
cd the_logits
```

### 2. Backend Initialization
```bash
cd backend
npm install
npm run dev
```

### 3. Frontend Deployment
```bash
cd frontend
npm install
npm run dev
```

### 4. Optional: Local LLM (Ollama)
Ensure Ollama is installed and running `llama3` on port 11434 for the full SYBIL experience.
